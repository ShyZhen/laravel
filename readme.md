## Environment

 - PHP >= 5.6.4
 - Apache
 - Mysql
 - Redis
 - Nodejs
 - ElasticSearch = 6.2.4 && ElasticSearch-analysis-ik (one index,one type)

## Installation

 - `git clone https://github.com/ShyZhen/laravel.git`
 - `copy .env.example .env` and edit .env
 - `composer install`
 - `npm install`
 - `php artisan migrate`  
 - `gulp`
